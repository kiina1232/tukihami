# 月喰み v0.3

5x5の点を印刷レーンでつないだ盤面で、月の移動によって石の明暗を反転させる2人用アブストラクトゲームです。

## 遊び方

- 黒と白が交互に手番を行います。
- 自分の手番では、月と同じ印刷レーン上の空き点に石を置きます。
- レーン上に置ける場所がない時だけ、任意の空き点に置けます。
- 月は置いた石へ移動します。
- レーン移動では、出発地点を除き、途中点と到着点の石を反転します。
- 最後に、月が止まった石は必ず影石になります。
- 月が乗っている石は、勝利判定では常に影石として扱います。
- 自分の明石を同じレーン上に3個連続させたら勝ちです。

## 勝利判定

月の移動と反転が終わったあとに判定します。

1. 手番プレイヤーに明石3連続があれば、手番プレイヤーの勝ち。
2. 手番プレイヤーに勝ちがなく、相手に明石3連続があれば、相手の勝ち。
3. どちらもなければ続行。

黒石12個、白石12個をすべて使い切って勝者がいない場合は、以下で判定します。

1. 明石の数が多い方が勝ち。
2. 同数なら、中央3x3にある明石が多い方が勝ち。
3. それも同じなら引き分け。

## オンライン対戦

オンライン対戦はWebRTCの手動コード交換です。サーバー、データベース、有料APIは使いません。

- 作成側は黒です。
- 参加側は白です。
- 作成側がオファーコードを送り、参加側がアンサーコードを返します。

## GitHub Pagesへの置き方

1. GitHubで公開リポジトリを作ります。
2. `index.html`, `README.md`, `.nojekyll` をリポジトリのルートに置きます。
3. `Settings` > `Pages` を開きます。
4. `Deploy from a branch` を選びます。
5. Branchを `main`、Folderを `/root` にして保存します。
6. `https://<user-name>.github.io/<repository-name>/` を開きます。

## 実装メモ

この版では、5x5正方グリッドだと終局判定に寄りすぎる問題を避けるため、横・縦・斜めの印刷レーン盤面に変更しています。端や角にも複数方向から月が入りやすくなり、勝利条件も明石3連続に短縮しました。

戦略が読みやすくなるように「出発地点は反転しない」と固定しています。到着石は反転処理のあとに必ず影石へ戻るため、置いた石は即座には勝利判定に使えません。
